# /deal

Deal structuring, IC memos, and transaction execution for PE, VC, and M&A professionals.

## Available Capabilities

| Area | What it does | Example prompt |
|------|-------------|----------------|
| IC Memos | Investment committee memos with standardized structure | "Draft an IC memo for this acquisition opportunity" |
| LBO Modeling | Leveraged buyout analysis with returns sensitivity | "Build an LBO model with 4x leverage and 5-year hold" |
| Term Sheets | Term sheet drafting and negotiation point analysis | "Draft a Series A term sheet with standard protections" |
| Exit Planning | Exit strategy development, timing, and buyer mapping | "Map potential exit routes for this portfolio company" |
| Deal Structuring | Transaction structure, tax optimization, earnout design | "Structure a management buyout with seller financing" |
| Process Management | Deal timeline, workstream tracking, closing checklists | "Create a 90-day deal execution timeline" |

## Quick Start

1. "Help me draft an IC memo for a 50M acquisition of a logistics company."
2. "Model an LBO with these assumptions and show me the IRR sensitivity table."
3. "What should the key term sheet provisions be for a growth equity deal?"
