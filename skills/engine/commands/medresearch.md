# /medresearch

Clinical trial design, grant writing, and biostatistics for medical researchers.

## Available Capabilities

| Area | What it does | Example prompt |
|------|-------------|----------------|
| Clinical Trial Design | Protocol development, endpoint selection, study architecture | "Design a Phase II trial for a novel oncology compound" |
| Grant Writing | NIH, ERC, and Horizon Europe grant applications and budgets | "Help me structure an R01 grant application for this study" |
| Biostatistics | Sample size calculations, statistical analysis plans, power analysis | "Calculate the sample size for a non-inferiority trial" |
| Literature Review | Systematic review frameworks, PRISMA methodology, evidence synthesis | "Set up a systematic review protocol for this research question" |
| Regulatory Strategy | FDA, EMA submission planning, IND/CTA preparation | "Outline the regulatory pathway for a Class III medical device" |
| Manuscript Preparation | Journal selection, structure, methods reporting standards | "Help me structure a manuscript for a randomized controlled trial" |

## Quick Start

1. "Design a randomized controlled trial protocol for a diabetes intervention."
2. "Write the specific aims section for an NIH R01 grant."
3. "Create a statistical analysis plan for a multi-center study."
