# /dev

Architecture design, testing strategies, CI/CD pipelines, and code review for full-stack developers.

## Available Capabilities

| Area | What it does | Example prompt |
|------|-------------|----------------|
| Architecture | System design, tech stack selection, scalability planning | "Design the architecture for a real-time collaboration platform" |
| Testing | Test strategy, unit/integration/E2E frameworks, coverage goals | "Set up a testing strategy for a Next.js application" |
| CI/CD | Pipeline design, deployment automation, environment management | "Build a GitHub Actions pipeline with staging and production" |
| Code Review | Quality analysis, pattern detection, refactoring suggestions | "Review this module for performance and maintainability" |
| Database Design | Schema modeling, migration strategies, query optimization | "Design a database schema for a multi-tenant SaaS application" |
| API Design | REST/GraphQL design, versioning, documentation standards | "Design a RESTful API for a marketplace platform" |

## Quick Start

1. "Help me design the architecture for a new microservices project."
2. "Set up a comprehensive testing strategy with Jest and Playwright."
3. "Review this codebase and suggest architectural improvements."
