# Premium Domains

Leopoldo's open-source platform includes the full system: orchestrator, agents, quality gates, correction loop, and auto-evolution. Premium domains add deep vertical expertise on top of that foundation, scoped to specific professional fields.

## What every premium domain includes

Every domain ships the complete system, not just the expertise layer.

| Component | Description |
|-----------|-------------|
| Orchestrator | Routes tasks to the right workflow agent for that domain |
| Workflow agents | Specialized agents for each major use case in the domain |
| Quality gates | Phase gates, doc gates, security gates. Blocking, not advisory. |
| Correction loop | Postmortem fires on every user correction. Root cause logged. |
| Evolution cycle | Weekly improvement cycle with user-approved patches |
| Auto-updates | New versions delivered automatically via GitHub Releases |

Expertise is additive. The system is always included.
