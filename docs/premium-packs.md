# Premium Plugins

Leopoldo's open-source platform includes the full system: orchestrator, agents, quality gates, correction loop, and auto-evolution. Premium plugins add deep vertical expertise on top of that foundation, scoped to specific professional fields. Each plugin ships with domain-specific agents that orchestrate multi-step workflows automatically.

## What every premium plugin includes

Every plugin ships the complete system, not just the expertise layer.

| Component | Description |
|-----------|-------------|
| Orchestrator | Routes tasks to the right workflow agent for that plugin |
| Workflow agents | Specialized agents for each major use case in the plugin |
| Quality gates | Phase gates, doc gates, security gates. Blocking, not advisory. |
| Correction loop | Postmortem fires on every user correction. Root cause logged. |
| Evolution cycle | Weekly improvement cycle with user-approved patches |
| Auto-updates | New versions delivered automatically |

Expertise is additive. The system is always included.
